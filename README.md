# funny-chuck

A simple people contacts searching application builded with [RANDOM USER GENERATOR](https://randomuser.me/) API using React.js, Material-UI.

### Features

- Displaying Results
- Searching
- HOC(Higher Order Component) Loader
- Material Design
- Responsiveness

## Install-Guide

```
> git clone https://github.com/user1990/people-finder-app.git
> cd people-finder-app
> npm install
> npm start
```

### Not Familiar with Git?
You can just download the .zip file. Extract the contents, then move to the project directory, open your terminal, and use:
```
> npm install
> npm start
```
